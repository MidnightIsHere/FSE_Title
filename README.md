# Title_Pack_For_FSE V1.0
这个仓库用于存储FSE服务器所使用的头衔系统  
目前的头衔系统版本是V1.0.1

V1.0.0  
主要包体制作完毕

V1.0.1  
修复了若干BUG，包括/team join相应队伍时未正确添加tag,/team leave时未正确移除tag以致于无法变更头衔为[玩家]  
缩短了扫地系统的工作冷却，从1000Tick/次变为800Tick/次

V1.0.2  
添加了许多成就，用于更好的游玩  
添加了许多合成表，大多数为武器  
将不会上传，因为合成表运用了外部模组  
你需要安装Create和Kubejs以解锁完全体  

V1.0.3  
使用Kubejs重构部分合成以防止误操作和降低难度  
重构的部分将包含在Kubejs文件夹中  
下载KubeJS.zip获取kubejs自定义物品  
下载[1.20+]Title_Pack_For_FSE.zip获取主要数据包  
解压kubejs文件夹并替换(如果你有自定义物品，请选择自主迁移)你的kubejs文件夹加载自定义物品  

V1.0.4  
预计加入多种提高生活质量的物品合成  
基于战利品表的遗迹物品箱合成  
kubejs更新方块  
三叉戟合成  

v1.0.5  
添加秘银，艾德曼和永恒合金，以及它们的合成方式  
配套装备和工具，锻造模板业已添加
